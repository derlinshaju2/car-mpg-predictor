# 🏎️ Car MPG Predictor
> A data-driven application exploring the **inverse relationship** between engine power and fuel economy.

## 📖 Overview
This project demonstrates **Negative Correlation** in Simple Linear Regression. Unlike many models where the target increases with the input, this app shows how Fuel Efficiency (MPG) decreases as Horsepower increases.

## ✨ Key Features
* **Interactive Slider:** Adjust the horsepower from 50 to 300 to see immediate MPG changes.
* **Regression Line Visualization:** View the "best fit" line through historical car data.
* **Prediction Markers:** Your current selection is highlighted in real-time on the scatter plot.

## 🛠️ Built With
* **Python**
* **Streamlit** (UI)
* **Scikit-Learn** (Machine Learning)
* **Matplotlib** (Data Viz)
